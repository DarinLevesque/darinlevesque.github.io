Month 2 Month Portraits

This unique photo series features a new born along their first year journey and typically has a prop that can be used for reference. Some of these were on location while others were done in studio and photoshopped to match the season. Many families will take this and run with it coming up with highly creative ways to photo document their childs first year.

Journey of Birth

Journey of Birth is a simple site for a doula business. It is built using Node.js with an express framework on top with Handlebars templating engine. It's all tied together with grunt and pushed to a remote git respository in a Dokku (docker clone) for ease of management.

  • Fields: Web,
  • Client: JOB Doula
